# flashcard_jp_v1
Flashcards for familiarizing yourself with and memorizing Japanese vocabulary.

To use it, paste the vocabulary list you want to learn into the input area, then start a round to generate flashcards automatically. Practice in either direction by viewing Japanese and recalling English, or viewing English and recalling Japanese, and review the usage notes as you go. After each card, score your recall using the buttons (for example, Easily recalled, Recalled with effort, or Forgot). Cards marked Forgot will be queued to appear again before the round ends. When you finish the round, review the summary table to see how you did and what to focus on next.
